# Golf Scorekeeper (Web App)

A mobile-first golf-scorekeeping Progressive Web App for IST 402.  
Built with **HTML, CSS, and JavaScript**, featuring undo/redo, draw logic, confetti celebrations, offline support, and local persistence.

**Live Site:** https://dlw5963.github.io/Golf-Scorekeeper/
